# Guia de Ejercicios Interactivos

Usa este archivo como hub rapido para practicar despues de cada capitulo.

## Plantilla de practica por capitulo
1) **Repaso rapida:** explica el concepto con tus palabras en 3 frases.  
2) **Mini reto de codigo:** escribe un snippet o modifica el ejemplo dado.  
3) **Reflexion:** que fallaria si cambias X? que suposicion es mas debil?  
4) **Extension:** como aplicarias esto en tu contexto personal o laboral?

## Sugerencias concretas
- Introduccion: cambia ejemplos de spam por mensajes de tu bandeja real (sin datos sensibles).  
- Fundamentos: calcula gradientes a mano para otra funcion sencilla.  
- Redes: agrega dropout en el MLP y mide perdida.  
- Deep Learning: implementa una capa conv1d en NumPy y prueba con señal simple.  
- Transformers: imprime la matriz de atencion y marca el mayor peso por fila.  
- LLMs: compara temperature 0.2 vs 1.0 en la misma pregunta.  
- RAG: varia k=1..5 y registra precision percibida.  
- Fine-tuning: diseña 20 ejemplos de tu dominio y define metricas de calidad.
